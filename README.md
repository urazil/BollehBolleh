# Bolleh Bolleh ? 작품 소개
```
BollehBolleh는 사용자가 동영상 및 사진들을 공유하는 어플입니다.
YouTube어플을 모티브로 만들었으며 동영상 게시물과 사진 게시물을 보고 댓글도 쓰면서 소통하는 어플입니다.
```

# 개발 환경 
* Android(Kotlin)
* Node.js
* MySQL
* Retrofit2
* AWS(RDS)
